URL Shortener API

Description:

This API allows you to create shortened urls by pasting a full url.

Example:

Enter:

https://cyan-suit.glitch.me/new/:urlname

And you will receive a JSON object like below:

{ "original_url":"http://foo.com:80", "short_url":"hhttps://cyan-suit.glitch.me/2030" }

You can then use the short_url in your browser and it will redirect you to the original url!
